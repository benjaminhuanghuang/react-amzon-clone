# How to Make an AMAZON Clone with REACT JS for Beginners (in 4 hours)

## Steps
- Setup app using
```
  npx create-react-app amazon-clone
```

- Setup firebase
```
  create a project

  npm i -g firebase-tools
```

- React Router
```
  npm i -S react-router-dom
```

- Nav bar + Material UI 
```
  npm i -S @material-ui/core
  npm i -S @material-ui/icons

```
